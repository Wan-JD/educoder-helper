头歌实践教学平台助手

## 简短描述（@description / 摘要）

解除头歌实训编辑器复制粘贴限制；支持一键复制代码、导出关卡文件、汇总测试用例、实验截止提醒。

## 附加信息 / 完整说明（粘贴到「附加信息」或脚本介绍）

### 适用网站

- https://www.educoder.net/*
- https://data.educoder.net/*（API 域名）

适用于头歌（Educoder）**教学课堂、课堂实验、图文作业、编程实训关卡**等所有使用 Monaco 在线编辑器的页面，**与学校无关**，任意高校账号均可使用。

### 主要功能

**1. 复制 / 粘贴解禁（核心）**

头歌部分作业会拦截 `Ctrl+C/V`、右键菜单，或在复制时写入空白内容。本脚本在页面最早阶段拦截这些限制，并持续修复 Monaco 编辑器的只读属性。

**2. 实训页悬浮面板（右下角 ⚡）**

进入「查看实战」后的关卡页可用：

| 按钮 | 作用 |
|------|------|
| 复制当前编辑器全部代码 | 方便粘贴到本地 VS Code / CLion 调试 |
| 导出关卡文件 | 将平台仓库中的多个源文件合并复制到剪贴板 |
| 汇总本页测试用例并复制 | 自动抓取左侧「测试输入」段落（编译、数据结构等课程很实用） |
| 重新解除粘贴限制 | 偶发失效时手动再应用一次 |

**3. 课堂实验截止提醒**

在「课堂实验」列表页顶部显示各实验剩余时间，不足 3 天会高亮。

**4. 公告通知（可选）**

在「通知公告」页检测到新公告时，尝试弹出浏览器通知（需授权）。

### 常见问题

**Q：安装后仍无法粘贴？**  
A：先用鼠标在代码编辑区点一下，再按 `Ctrl+V`。仍不行则点击 ⚡ →「重新解除粘贴限制」。

**Q：导出文件是空的？**  
A：请先在编辑器左侧文件树中点开过要导出的文件，让平台加载过 `rep_content` 接口后再点导出。

**Q：会不会被封号？**  
A：脚本仅在前端解除 UI 限制、读取你已登录可见的数据，不提交答案、不攻击平台。请仍遵守课程与考试规定。

### 不包含的功能（刻意不做）

- 自动评测刷分、自动填标准答案  
- 任何违反学术诚信的自动化
