# 头歌实践教学平台助手

适用于 [educoder.net](https://www.educoder.net)（头歌实践教学平台）的 Tampermonkey 用户脚本。

## 功能

- **解除复制 / 粘贴 / 全选限制**：在课堂实验、图文作业、Monaco 编辑器等页面恢复 `Ctrl+C/V/X/A`
- **实训页工具面板**（右下角 ⚡）：
  - 一键复制当前编辑器全部代码
  - 导出关卡多文件到剪贴板（基于平台 `rep_content` 接口）
  - 汇总本页「测试输入」用例并复制（适合编译原理等有多组测例的课程）
- **课堂实验列表**：显示实验截止倒计时，3 天内标红提醒
- **通知公告**：有新公告时尝试桌面通知（需浏览器权限）

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/)
2. 从 [Greasy Fork](https://greasyfork.org/) 安装，或安装开发版：

   ```
   https://github.com/Wan-JD/educoder-helper/raw/main/educoder-helper.user.js
   ```

3. 登录头歌后，进入任意课堂或实训「查看实战」页面，右下角应出现 **⚡** 按钮

## 使用说明

| 场景 | 操作 |
|------|------|
| 编辑器无法粘贴 | 先点击代码区获得焦点，再 `Ctrl+V`；或打开 ⚡ 面板点「重新解除粘贴限制」 |
| 本地 IDE 写完再提交 | ⚡ →「复制当前编辑器全部代码」，或反向粘贴进编辑器 |
| 多文件实验 | 在编辑器中切换过各文件后，⚡ →「导出关卡文件」 |
| 测例太多不想手抄 | 在关卡任务描述页，⚡ →「汇总本页测试用例并复制」 |
